type cases = [
  Expect<Equal<BEM<'btn', ['price'], []>, 'btn__price'>>,

  BEM是CSS中类的流行的命名规范.

  例如, 用`btn`表示块元素, 依赖于块元素的将用`btn__price`表示, 改变块样式的修饰符适用`btn--big`或`btn__price--warning`来修饰.
  执行`BEM<B,E,M>`,从3个参数中生成联合字符串. B是字符字面量, E和M是字符数组(也可能是空的).
