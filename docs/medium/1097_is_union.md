实现一个类型IsUnion, 接收一个类型T, 如果解析为联合类型则返回true, 否则返回false

  例如：

  ```ts
  type case1 = IsUnion<string> // false
  type case2 = IsUnion<string | number> // true
  type case3 = IsUnion<[string | number]> // false
  ```
